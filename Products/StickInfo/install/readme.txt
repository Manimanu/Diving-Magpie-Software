This program simply queries the abilities of the
first installed joystick driver.

It does *not* confirm that the joystick is actually 
functioning correctly, it merely checks that the
driver is correctly installed.

If you want to check if the joystick is working
properly, this is best done through the calibration
procedure of the Windows control panel.
