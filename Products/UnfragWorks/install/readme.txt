Very Important!!
Please make sure you have the correct version 
of UnfragWorks for your operating system.  There
are separate versions for Windows NT and 
Windows 95.  They are not interchangeable.

------------------------------------------------------------------------

Installation:
Several entries have to be made in the registry.
It is highly recommended that this be left to 
the installation program.

For the brave of heart attempting a manual
installation, please read the Manual
Installation section at the end.

------------------------------------------------------------------------

Common Problems:

1) UnfragWorks doesn't work:
For those people doing a manual installation,
there are separate version of UnfragWorks for
Windows NT and Windows 95.  They are NOT
interchangeable.  Please make sure you are 
using the correct version.

2) UnfragWorks still doesn't work:
For those people doing a manual installation,
make sure the UnfragWorks.exe is located
in the Windows directory.

------------------------------------------------------------------------

Manual Installation:
1) The following files are required:
    All platforms:
	UnfragWorks.exe         UnfragWorks
	UnfrgWks.hlp            help file
	UnfragWorks.ini         configuration file
	SldPrt.reg              registry settings
	SldAsm.reg              registry settings
	SldDrw.reg              registry settings
	IsUn32ex.exe            uninstallation program

    Windows NT:
	UnfragWorksNT.reg       registry settings
	UnfragWorksNT.dll       UnfragWorks addin

    Windows 95:
	UnfragWorks95.reg       registry settings
	UnfragWorks95.dll       UnfragWorks addin

2) Place all the UnfragWorks files, except for
UnfragWorks.exe & UnfragWorks.ini, in the:

    C:\Program Files\SolidPartners\UnfragWorks

directory.  If this directory does not exist, create it.

Place UnfragWorks.exe & UnfragWorks.ini in your 
Windows directory, usually C:\WinNT (Windows NT) 
or C:\Windows (Windows 95).

3) Double click on the registry settings files:
	SldPrt.reg
	SldAsm.reg
	SldDrw.reg

and

	UnfragWorksNT.reg
or

	UnfragWorks95.reg

as appropriate, to install them to the Windows registry.

------------------------------------------------------------------------

Manual Uninstallation:
1) run the unistallation program IsUn32ex.exe to remove the
registry settings

2) delete the program, help and configuration files

3) delete the installation directory
