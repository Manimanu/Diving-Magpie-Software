Manual Installation:
1) The following files are required:
    Windows NT:
	PictureWorksNT.reg      registry settings
	PictureWorksNT.dll      PictureWorks
	PictWks.hlp             help file
	IsUn32ex.exe            uninstallation program

    Windows 95:
	PictureWorks95.reg      registry settings
	PictureWorks95.dll      PictureWorks
	PictWks.hlp             help file
	IsUn32ex.exe            uninstallation program

2) Place all the PointerWorks files, except for
PtrWorks.ini, in the:

    C:\Program Files\SolidPartners\PictureWorks

directory.  If this directory does not exist, create it.

3) Double click on the registry settings file
to install it to the Windows registry.

------------------------------------------------------------------------

Manual Uninstallation:
1) run the unistallation program IsUn32ex.exe to remove the
registry settings

2) delete the program, help and configuration files

3) delete the installation directory
