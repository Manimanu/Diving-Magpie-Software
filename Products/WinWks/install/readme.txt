Installation:
Several entries have to be made in the registry.
It is highly recommended that this be left to 
the installation program.

For the brave of heart attempting a manual
installation, please read the Manual
Installation section at the end.

------------------------------------------------------------------------

Common Problems:

------------------------------------------------------------------------

Manual Installation:
1) The following files are required:
    Windows NT:
	WinWksNT.dll		WindowWorks
	WinWksNT.reg		registry file
	IsUn32ex.exe            uninstallation program

    Windows 95:
	WinWks95.dll		WindowWorks
	WinWks95.reg		registry file
	IsUn32ex.exe            uninstallation program

    Windows NT & 95 system:
	WinWks.hlp		help file
	WinWks.dll		BundleWorks support file

2) Place all the WindowWorks files, except for
the system files, in the:

    C:\Program Files\SolidPartners\WindowWorks

directory.  If this directory does not exist, create it.

Place the sysem files in your Windows system directory, 
usually C:\WinNT\system32 (Windows NT) or 
C:\Windows\system (Windows 95).

3) Double click on the registry settings files
to install them to the Windows registry.

------------------------------------------------------------------------

Manual Uninstallation:
1) run the uninstallation program IsUn32ex.exe to remove 
the registry settings

2) delete the program, help and configuration files

3) delete the installation directory
