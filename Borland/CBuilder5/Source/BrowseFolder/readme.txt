***** Browse Folders Dialog for Borland C++ Builder 4.0 *****

The accompanying files implement the native Windows 9x Browse Folders dialog.  It is most useful for allowing a user to select a folder (not a specific file) for subsequent activities.  The code is implemented as a BCB 4.0 component.

You are free to use this code in any way you wish.  However, please do not share, post, or otherwise distribute this code.  Also, if you find bugs or make improvements, please let me know.

The most current version of this code can be found at http://home.att.net/~robertdunn/Yacs.html.
