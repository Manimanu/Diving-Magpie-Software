FreeImage 1.4.3

Thank you for downloading FreeImage, the Open Source image library for multimedia and game programmers. The goal of FreeImage is to make it one of the leading image loading and manipulation libraries, while keeping the Application Programming Interface (API) as easy to use as possible. A wide range of functions are provides to load BMPs, JPEGs, PNGs, etc and to property convert and manipulate them. This manual gives you insight in how the library works, so that you can include it in your programs.

If you have any questions about this manual, or about FreeImage, please don't hesitate to send an e-mail to mailto:freeimage@wxs.nl
